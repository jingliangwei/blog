# 实测天体物理 期末复习

## 绪论

1. 十九世纪中叶，三种物理方法（照相术、分光学、光度学）促进了天体物理学的发展

2. 2002年诺贝尔物理学奖 中微子天文学、X射线天文学

3. 2011年诺贝尔物理学奖 Ia型超新星->宇宙加速膨胀

4. 2017年诺贝尔物理学奖 LIGO探测引力波

5. 学科发展：天体测量学->天体力学->天体物理学

6. 现代天文学之父（一到六等星）喜帕恰斯

7. 星系天文学之父（河外星系）哈勃

8. 恒星天文学之父（双星、变星）赫歇尔

9. 肉眼时代最伟大的观测家 第谷

10. 第谷的观测贡献：
    
    1. 1572.11.11仙后座超新星爆发
    2. 精密观测资料（开普勒行星运动定律）
    3. 精密星表，大气折射、黄赤交角的变化和月亮运动的二均差，测定岁差常数

11. 伽利略的成就：
    
    1. 第一台天文望远镜
    2. 银河由无数单个恒星组成
    3. 月面不平整
    4. 太阳黑子位移推测太阳自转周期28天
    5. 金星有圆缺变化，土星环，木星四个卫星

12. 牛顿：反射望远镜

## 第一章

1. 五种天体信息载体：电磁辐射、物质采样、宇宙射线、引力波、中微子

2. 天体电磁辐射：

| 辐射类型 | 波长范围 | 温度($K$) | 典型辐射源 |
|:---:|:---:|:---:|:---:|
| $\gamma$ 射线 | $0.03-0.1\overset{\circ}{\mathrm{A}}$ | $>10^8$ | 吸积盘、恒星闪耀、黑洞、活动星系核 |
| X射线 | $0.1-100\overset{\circ}{\mathrm{A}}$ | $10^6-10^8$ | 星系团气体、超新星遗迹、恒星冕区、活动星系核 |
| 紫外 | $100-4000\overset{\circ}{\mathrm{A}}$ | $10^4-10^6$ | 超新星遗迹、热恒星 |
| 可见光 | $4000-7600\overset{\circ}{\mathrm{A}}$ | $10^3-10^4$ | 大部分恒星、星系 |
| 红外 | $0.76\mu m-1mm$ | $10-10^3$ | 冷尘埃云和气体、行星 |
| 射电 | $1mm-100km$ | $<10$ | 磁场中电子运动 |

3. 地球大气对天文观测的影响：
   1. 大气折射，改变天体辐射方向（蒙气差）
   2. 大气消光，改变天体辐射的成分和强度
   3. 大气辐射，增强天光背景，影响极限星等和观测信噪比
   4. 大气的反射和散射，光污染，增强天光背景，影响极限星等和观测信噪比
   5. 大气湍动，使星象变大变模糊（视宁度）

4. 三个大气窗口：光学窗口、红外窗口和射电窗口

5. 视宁度：天顶方向像斑的半峰全宽

6. 大气视宁度和星象半峰全宽区别：后者还包含衍射效应、望远镜光学质量、圆顶视宁度、星际闪烁、行星际闪烁等

7. 影响观测其他因素（除了大气）：
   1. 黄道：黄道面上由尘埃、小行星带
   2. 极光带：可以用滤光片过滤
   3. 银河消光：银河系的星际介质消光（两大消光：大气+银河）

8. 天文台选址评估参数：
   1. 普查参数：
      1. 气候情况：云量、年晴日数和年降水量
      2. 地理位置：海拔、交通、水电供应
      3. 光和大气污染：远离城市
      4. 地震活动情况
   2. 定点观测参数：
      1. 测光夜和测光小时（晴天数、云量）——使用效率
      2. 大气视宁度：成像质量的好坏
      3. 大气积分水汽含量：直接影响红外与亚毫米波段的观测
      4. 大气消光（散射）和天光背景（夜天光）：影响极限星等
      5. 气象学参数：风力，温度和温差、湿度等

## 第二章 望远镜

1. 一个完整的现代天文光学观测仪器系统：望远镜+辐射分析仪+辐射探测器+计算机

2. 望远镜光学性能参量：
   1. 口径
   2. 相对口径（光力） $A=D/F$ 焦比 $F/D$
   3. 视场
   4. 放大率（底片比例尺） $M=F_1/F_2$ $\alpha=206265/F(''/mm)$
   5. 贯穿本领（极限星等） $m=2.1+5\log D(mm)$
   6. 分辨本领 $\delta=1.22\dfrac{\lambda}{D}$
   7. 光能集中度: $80\%$ 或 $90\%$ 的光能集中在直径为多少角秒

3. 像差：
   1. 球差
   2. 慧差
   3. 像散
   4. 场曲
   5. 畸变
   6. 色差（位置色差、倍率色差）

4. 折射望远镜现在用的少原因：
   1. 大口径光学透镜制造困难，并且容易受温度、压力和重力影响变形
   2. 玻璃会吸收入射光，尤其是对蓝、紫光吸收严重
   3. 六种像差均比较严重，影响成像质量

5. 反射望远镜：没有色差，采用抛物面镜消球差

6. 反射望远镜焦点系统：
   1. 主焦点系统
   2. 牛顿焦点系统
   3. 卡塞格林焦点系统
   4. R-C焦点系统
   5. Nasmyth 焦点系统
   6. 折轴焦点系统

7. 施密特望远镜：主镜为球面镜，改正镜为旋转四次曲面镜（改正镜位于主镜球心处）

8. 马克苏托夫望远镜：改正镜为弯月形透镜
   
   优点：
   1. 光学系统所有表面均为球面，容易加工
   2. 改正镜放在主镜焦点附近，镜筒比施密特望远镜短

   缺点：
   1. 视场比同口径同焦比的施密特望远镜小，成像质量差
   2. 弯月形透镜的厚度要达到口径十分之一左右，光损较大，限制口径

9. 反射望远镜：（光度测量、分光观测）
   
   优点：
   1. 没有色差
   2. 光损小
   3. 口径可以很大
   4. 相对口径大。焦距小、镜筒短、观测室小、省钱
   5. 可以兼具多种焦点系统

   缺点：
   1. 镜面形状对温度敏感。（微晶玻璃、碳化硅改善）
   2. 镜面需要定期重新镀膜，1-2次/年，麻烦，风险
   3. 视场小，一般不超过 1 度

10. 折射望远镜：（天体测量、恒星天文）
   
    优点：
    1. 视场较大
    2. 放大率大
    3. 受温度变化影响小，星象稳定
    4. 采用镜筒，仪器散射光比反射望远镜小
    5. 采用镜筒，受圆顶视宁度影响小

    缺点：
    1. 透镜在边上支撑，受自重影响变形
    2. 需保证镜子中间无气泡
    3. 典型焦比 $F/15$ ，镜筒长，观测室大，造价高
    4. 对短波吸收严重，近紫外无法观测

11. 折反射望远镜：（巡天）

    优点：
    1. 焦距短，光力强， $F/D=1\sim2$ ，可以观测很暗的面源
    2. 视场大，可以达到 10 度以上

    缺点：
    1. 焦点在镜筒内部，终端设备挡光，操作不便
    2. 改正镜形状特殊，难以加工，尺寸有限

12. 望远镜的机械装置要求 指向精度和跟踪精度

13. 主动光学：矫正由重力、温度和风力造成的望远镜光学系统变化、主要是主镜形变

14. 自适应光学：跟踪和补偿大气湍流造成的光的波阵面畸变

## 第三章 辐射探测器

1. 探测器性能参量：
   1. 量子效率 QE：记录光子数比入射光子数
   2. 探测量子效率 DQE：输出信噪比的平方与输入信噪比的平方之比
   3. 响应度（灵敏度）$R$：输入信号和输出信号之比
   4. 光谱响应（分光灵敏度）$R_\lambda$：对不同波长的响应
   5. 分辨率：空间、时间分辨率
   6. 探测率：能探测到的最小辐射功率的倒数，可以用噪声等效功率 NEP 表示
   7. 暗噪声：没有输入时探测器的输出
   8. 动态范围：响应度呈线性的范围

2. 光电倍增管的优点 时间分辨率高

3. CCD 优点：
   1. 量子效率高
   2. 噪声低
   3. 动态范围宽
   4. 分光响应范围宽
   5. 线性度好

4. CCD 读出电子数，计算机显示信号单位 ADU$=e^-/G$, $G$ 为增益

5. 仪器星等 $m_{inst}=-2.5\log ADU+C$

6. CCD 读出方式：
   1. 逐行读出
   2. 帧转移读出
   3. 行间转移读出
   4. 漂移扫描读出
   5. 可变积分时间的漂移扫面读出
   6. 简并读出
   7. 局部读出
   8. 分块（多门）读出
